# KW dorm app

광운대학교 기숙사 공지사항을 조금 더 편하게 확인해보고 싶어서 개인적으로 개발을 시도한 프로젝트입니다.

## 프로젝트 소개

서버에서 기숙사 공지 페이지를 가져와 필요한 정보를 보여주는 기능을 구현해보는 것을 목표로 했습니다. 백엔드 개발을 처음 접하는 단계에서 학습 목적으로 시작했습니다.

## 진행 내용

* Node.js를 사용해 간단한 서버 구성
* 기숙사 공지 페이지 크롤링 기능 구현 시도
* 원하는 데이터를 가져오기 위해 여러 방식으로 선택자 수정 및 테스트 진행

## 사용 기술

* Node.js
* Express
* Axios
* Cheerio

## 어려웠던 점

크롤링 과정에서 원하는 데이터가 정상적으로 추출되지 않아 여러 번 수정과 테스트를 반복했습니다. 원인을 바로 찾기 어려워 요청 결과를 확인하고 선택자를 바꿔보며 문제를 좁혀가는 과정을 경험했습니다.

## 배운 점

처음 접하는 기술이라 구현이 쉽지 않았지만, 모르는 상태에서도 시도하며 흐름을 이해하는 경험을 할 수 있었습니다. 완전히 구현하지는 못했지만 서버와 크롤링이 어떻게 동작하는지 이해하는 계기가 되었고, 이후 백엔드 학습을 이어가고 싶다는 동기를 얻었습니다.
